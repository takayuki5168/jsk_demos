## curry

ロボットにカレーを作ってもらう。
最終的には、ご飯（無洗米）、サラダも作って夕食を作る。スープ（お湯を注ぐ方式？）とデザートにりんごの皮むきも出来たら良い。

## カレーの作り方
- 具材を切る  
- 具材を炒める  
- 水を加えて煮込む  
- ルーを加えて煮込む  

## 使う食材
- 豚肉  
- 人参  
- じゃがいも  
- 玉ねぎ  
- カレールー  
- サラダ油
